🛒 Flipkart Backend Clone (CRUD API)

A simple Flipkart-style backend project built using Node.js and Express.js, implementing full CRUD operations for Users and Products.

🚀 Features
✅ User Management (Create, Read, Update, Delete)
✅ Product Management (Create, Read, Update, Delete)
✅ RESTful API structure
✅ Organized folder structure (MVC / layered)
✅ JSON-based request & response handling
✅ Error handling & validation (basic)
🛠️ Tech Stack
Node.js
Express.js
MongoDB (or JSON / in-memory if used)
Mongoose (if MongoDB used)
